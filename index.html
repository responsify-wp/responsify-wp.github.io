Made for developers. To

With great powers comes great responsive websites. Responsify WP allows you to take control of your responsive images.

Handpick sizes

Select exactly which image sizes that RWP should use.

{% highlight php %} 'portfolio', 'rwp_settings' => array( 'sizes' => array('large', 'full') ) ) ); foreach ( $posts as $post ) { // ... } ?> {% endhighlight %}

Media queries

Specify which image size that should be used at which viewport size.

{% highlight php %} 'portfolio', 'rwp_settings' => array( 'sizes' => array('thumbnail', 'medium', 'large'), 'media_queries' => array( 'medium' => 'min-width: 500px', 'large' => 'min-width: 1024px' ) ) ); $query = new WP_Query( $args ); ?> {% endhighlight %}
